b. NPM (Node Package Manager)
What is NPM?

NPM is the default package manager for Node.js.
It allows developers to install and manage JavaScript packages.

ğŸ‘‰ Example command:

npm install express

Why is NPM important for Node.js applications?

NPM is important because it:

Provides access to millions of open-source packages

Automatically manages dependencies

Helps in running scripts

Ensures consistency across environments

How NPM helps in managing dependencies

Stores dependency information in package.json

Locks exact versions in package-lock.json

Automatically installs required sub-dependencies

ğŸ‘‰ Example:

"dependencies": {
  "express": "^4.18.2"
}


c. Backend Project Initialization
What is the command used to initialize a backend (Node.js) project?
npm init

Explain npm init

Asks questions (project name, version, description, etc.)

Creates a package.json file

Allows customization of project metadata

Explain npm init -y

Skips all questions

Automatically creates package.json with default values

Faster project setup
